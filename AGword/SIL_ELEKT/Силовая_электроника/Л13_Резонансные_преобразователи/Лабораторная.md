# Лабораторная 13. Резонансный преобразователь (LLC)

**Тема:** Моделирование LLC, мягкая коммутация (ZVS), частотное регулирование.

## Цель
Исследовать работу резонансного преобразователя: подтвердить ZVS, снять характеристику усиления от частоты, сравнить потери с жёсткой коммутацией.

## Теория и расчёт (до симуляции)
```
f₀ = 1/(2π·√(L_r·C_r)),   Z₀ = √(L_r/C_r),   Q = Z₀/R_экв
```
Дано: `L_r = 100 мкГн`, `C_r = 22 нФ`.
```
f₀ = 1/(2π√(100e-6·22e-9)) ≈ 107 кГц
Z₀ = √(100e-6/22e-9) ≈ 67 Ом
```
Намагничивающая `L_m = 5·L_r = 500 мкГн`.

---

## Часть A. Схема (LTspice)

### A1. Полумост + резонансный контур
1. Питание `V_dc = 400 В`. Полумост из двух MOSFET (комплементарное управление, ~50% скважность, мёртвое время 200 нс).
2. От средней точки полумоста: `C_r = 22 нФ` → `L_r = 100 мкГн` → первичка трансформатора (`L_m = 500 мкГн`).
3. Трансформатор `N_p:N_s = 4:1`, вторичка → мостовой выпрямитель → `C = 100 мкФ` → нагрузка `R = 10 Ом`.
4. Управление: два `PULSE` в противофазе, частота `f_sw` — параметр.

### A2. Параметрический прогон по частоте
`.step param fsw 75k 140k 5k` — прогон частоты вокруг f₀.
`.tran` для каждого значения; снимите установившееся `U_вых`.

---

## Часть B. Измерения

### B1. Характеристика усиления
1. Для `f_sw` = 80, 95, 107, 120, 140 кГц снимите `U_вых`.
2. Постройте `U_вых(f_sw)` — колоколообразная кривая с пиком у f₀.
3. Рабочую точку выберите на спадающем участке **выше** f₀ (индуктивная зона — там ZVS).

### B2. Подтверждение ZVS
1. Снимите `V(сток)` и `V(затвор)` верхнего ключа.
2. Убедитесь: напряжение сток-исток спадает до нуля **до** подачи импульса на затвор (включение при нуле напряжения).
3. Отметьте, что ток через ключ при включении отрицателен (течёт через body-diode) — признак ZVS.

### B3. Сравнение потерь
Сравните потери переключения с жёстким Buck (ЛР-09) при той же мощности. У LLC `E_on ≈ 0` (ZVS).
