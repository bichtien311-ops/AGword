# Лекция 05. Неуправляемые выпрямители

**Модуль II. Преобразователи AC/DC**

---

## 1. Задача выпрямления

Выпрямитель преобразует переменное напряжение сети в постоянное (точнее — в пульсирующее однополярное, которое затем сглаживается фильтром). «Неуправляемый» означает, что используются только диоды — выходное напряжение не регулируется, оно жёстко задано амплитудой сети и схемой.

Качество выпрямителя оценивают по:
- среднему выходному напряжению `U_ср`;
- коэффициенту пульсаций `k_п`;
- частоте пульсаций (чем выше, тем легче фильтровать);
- нагрузке на диоды (средний и действующий ток, обратное напряжение).

---

## 2. Однополупериодный выпрямитель

Один диод пропускает только положительные полупериоды. Среднее значение:
```
U_ср = (1/2π) ∫₀^π U_m·sin θ dθ = U_m/π ≈ 0.318·U_m
```
**Недостатки:** очень большие пульсации (`k_п ≈ 1.21`), частота пульсаций = частоте сети (50 Гц), подмагничивание трансформатора постоянной составляющей. На практике почти не используется в силовых цепях.

---

## 3. Однофазный мостовой выпрямитель (схема Гретца)

### 3.1. Принцип

Четыре диода образуют мост. В каждый полупериод проводит **пара диодов** по диагонали, так что ток через нагрузку всегда течёт в одном направлении, используя **оба полупериода**.

### 3.2. Вывод среднего напряжения

Выходное напряжение — это `|U_m·sin θ|`, период пульсаций равен **половине** периода сети. Среднее:
```
U_ср = (1/π) ∫₀^π U_m·sin θ dθ = (U_m/π)·[−cos θ]₀^π = 2·U_m/π ≈ 0.637·U_m
```
Через действующее: `U_m = √2·U_rms`, поэтому
```
U_ср = 2√2/π · U_rms ≈ 0.9·U_rms
```

### 3.3. Пульсации

Частота пульсаций — **2f** (100 Гц при сети 50 Гц). Коэффициент пульсаций по первой гармонике `k_п ≈ 0.48` (значительно меньше однополупериодного).

### 3.4. Параметры диодов
- Среднее значение тока диода = половина тока нагрузки (`I_ср/2`).
- Обратное напряжение на закрытом диоде ≈ `U_m`.

---

## 4. Трёхфазный мостовой выпрямитель (схема Ларионова)

### 4.1. Принцип

Шесть диодов, по два на каждую фазу (верхняя и нижняя группа). В любой момент проводят два диода: один из верхней группы (фаза с наибольшим потенциалом) и один из нижней (фаза с наименьшим). Выходное напряжение «следует» за огибающей линейных напряжений.

### 4.2. Вывод среднего напряжения

За период сети происходит **6 пульсаций** (частота 6f = 300 Гц). Выходное напряжение на интервале проводимости пары — отрезок линейного напряжения `U_лm·sin θ` вблизи максимума, в пределах `θ ∈ [π/3, 2π/3]`. Среднее по одному 60°-интервалу:
```
U_ср = (3/π) ∫_{π/3}^{2π/3} U_лm·sin θ dθ = (3·U_лm/π)·[cos(π/3) − cos(2π/3)]
     = (3·U_лm/π)·(0.5 + 0.5) = 3·U_лm/π
```
где `U_лm` — амплитуда **линейного** напряжения. Через действующее линейное `U_л`:
```
U_ср = 3√2/π · U_л ≈ 1.35·U_л
```

### 4.3. Малые пульсации

Так как выход «срезает» только верхушки синусоид, размах пульсаций мал. Коэффициент пульсаций `k_п ≈ 0.057` (5.7%) — на порядок лучше однофазного. Поэтому трёхфазные мосты доминируют в мощных промышленных выпрямителях.

---

## 5. Влияние характера нагрузки

### 5.1. Активная (R)
Ток повторяет форму напряжения; всё просто.

### 5.2. Индуктивная (RL, большая L)
Индуктивность сглаживает ток — он становится почти постоянным. Ток через диоды — прямоугольные импульсы. Это благоприятный режим: непрерывный ток нагрузки.

### 5.3. Ёмкостная (RC, конденсатор на выходе)
Конденсатор заряжается до амплитуды только вблизи пиков напряжения, а в остальное время разряжается на нагрузку. Диоды проводят **короткими импульсами большого тока** (пиковый ток >> среднего).

**Описание графика тока диода при C-нагрузке:** узкие высокие импульсы тока вблизи каждого пика сетевого напряжения, между ними — ноль. Это повышает действующий ток диодов и ухудшает коэффициент мощности (см. лекцию 8).

---

## 6. Численные примеры

**Однофазный мост, `U_rms = 220 В`:**
```
U_m = 220·√2 ≈ 311 В,  U_ср = 2·311/π ≈ 198 В
```

**Трёхфазный мост, `U_л = 380 В`:**
```
U_ср ≈ 1.35·380 ≈ 513 В
```

---

## 7. Выводы

1. Однополупериодная схема непрактична (большие пульсации, подмагничивание).
2. Однофазный мост: `U_ср ≈ 0.9·U_rms`, пульсации 2f, `k_п ≈ 0.48`.
3. Трёхфазный мост: `U_ср ≈ 1.35·U_л`, пульсации 6f, `k_п ≈ 5.7%` — выбор для мощных систем.
4. Характер нагрузки кардинально меняет форму токов; ёмкостная нагрузка создаёт пиковые токи диодов.
