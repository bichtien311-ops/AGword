# Лекция 12. Трёхфазные инверторы: SPWM и SVPWM

**Модуль III. Преобразователи DC/DC и DC/AC**

---

## 1. Трёхфазный мостовой инвертор

Три стойки по два ключа (всего 6 ключей с антипараллельными диодами), питаемые от общей DC-шины. Средние точки стоек (A, B, C) подключаются к трёхфазной нагрузке (звезда или треугольник). Каждая стойка формирует одно фазное напряжение, сдвинутое на 120° относительно других.

Основное применение — частотно-регулируемый электропривод и сетевые инверторы.

---

## 2. Режим 180°-проводимости (прямоугольное управление)

Каждый верхний ключ открыт полпериода (180°), стойки сдвинуты на 120°. Выходные потенциалы — меандры со сдвигом 120°. **Линейное напряжение** получается вычитанием потенциалов стоек и имеет ступенчатую форму (six-step). Спектр содержит крупные гармоники (5, 7, 11, 13-я; кратные 3 в линейном напряжении отсутствуют). Применяется при простом управлении, но качество тока низкое.

---

## 3. Синусоидальная ШИМ для трёх фаз (SPWM)

Три синусоиды-задания, сдвинутые на 120°, сравниваются с **общей** треугольной несущей. Каждая стойка формирует ШИМ-напряжение, среднее значение которого следует за своей синусоидой.

### 3.1. Амплитуды напряжений

Амплитуда **фазного** напряжения (относительно средней точки шины):
```
U_ф1m = m_a·U_dc/2     (m_a ≤ 1)
```
Амплитуда **линейного** напряжения (между фазами) больше в `√3`:
```
U_л1m = √3·U_ф1m = (√3/2)·m_a·U_dc
```

### 3.2. Предельное использование DC-шины

При `m_a = 1` максимальная амплитуда линейного:
```
U_л1m(max) = (√3/2)·U_dc ≈ 0.866·U_dc
```
То есть SPWM использует лишь ~86.6% потенциала DC-шины. Остальное «недоступно» без перемодуляции.

---

## 4. Векторная ШИМ (SVPWM)

### 4.1. Идея

Три выходных потенциала рассматриваются как **единый вектор напряжения** в плоскости (α, β). Восемь возможных комбинаций состояний 6 ключей дают **6 активных векторов** (вершины шестиугольника) и **2 нулевых** (центр). Требуемый вектор синтезируется как **взвешенное по времени** сочетание двух соседних активных векторов и нулевых.

### 4.2. Преимущество по использованию шины

Геометрически: максимальная окружность, вписанная в шестиугольник активных векторов, имеет радиус больше, чем у SPWM. В результате достижимая амплитуда линейного напряжения:
```
U_л1m(max) = U_dc
```
Это на `2/√3 ≈ 1.155`, то есть **на ~15% больше**, чем у SPWM. Эквивалентно добавлению 3-й гармоники в задание (third-harmonic injection).

### 4.3. Нулевые векторы

Распределение времени между двумя нулевыми векторами (000 и 111) — степень свободы, которую используют для минимизации числа переключений и потерь. SVPWM удобен для цифровой реализации (DSP).

**Описание графиков:** фазное напряжение SVPWM имеет характерную «седловидную» форму огибающей (за счёт инжекции 3-й гармоники), но линейное напряжение остаётся синусоидальным. Спектр: основная гармоника + группы у частоты несущей; линейное «чище» фазного.

---

## 5. Мёртвое время (dead time)

Чтобы оба ключа одной стойки не оказались открыты одновременно (сквозной ток, КЗ шины), при переключении вводят **мёртвое время** — паузу, когда оба ключа закрыты. Платой за это являются искажения: появляются низкочастотные гармоники (5, 7-я) в выходном токе, особенно заметные при малой амплитуде. Их компенсируют программно (dead-time compensation).

---

## 6. Численный пример

`U_dc = 540 В`, `m_a = 1`:
- SPWM: `U_л1m = 0.866·540 ≈ 467 В` (амплитуда), RMS ≈ 330 В.
- SVPWM: `U_л1m ≈ 540 В` (амплитуда), RMS ≈ 382 В.

Выигрыш SVPWM по напряжению ~15% — это больше выходной мощности привода без увеличения шины.

---

## 7. Выводы

1. Трёхфазный мост (6 ключей) формирует три напряжения со сдвигом 120°.
2. SPWM: `U_л1m(max) ≈ 0.866·U_dc` — недоиспользует шину.
3. SVPWM синтезирует вектор из соседних базовых + нулевых, поднимая `U_л1m(max)` до `U_dc` (+15%).
4. Мёртвое время предотвращает сквозной ток, но вносит искажения, требующие компенсации.
