# Лабораторная 08. Анализ гармоник и активный PFC

**Тема:** Измерение THD и PF выпрямителя, моделирование Boost-PFC.

## Цель
Оценить качество потребляемого тока диодного выпрямителя с C-фильтром и показать улучшение при активной коррекции коэффициента мощности.

## Теория и расчёт (до симуляции)
```
PF = k_иск·cos φ₁,   k_иск = 1/√(1 + THD²)
```
- Выпрямитель + C: ожидаем `THD ≈ 80…120%` → `PF ≈ 0.5…0.7`.
- С PFC: `THD < 5%` → `PF ≈ 0.99`.

## Часть A. Выпрямитель с C-фильтром (Simulink)

### A1. Схема
1. `AC Voltage Source`: 311 В, 50 Гц.
2. `Universal Bridge` (Diodes, arms = 2).
3. `C = 470 мкФ` + нагрузка `R = 200 Ом`.
4. Измеритель тока сети `i_с` → в `powergui`.

### A2. FFT-анализ
1. `powergui → FFT Analysis`. Сигнал — ток сети.
2. Fundamental = 50 Гц, диапазон до 2 кГц.
3. Зафиксируйте `THD`, амплитуды 3, 5, 7-й гармоник.
4. Вычислите `PF` (через измеренные P и S или блок `Power (3ph instantaneous)`).

---

## Часть B. Активный Boost-PFC (Simulink)

### B1. Силовая часть
1. Диодный мост → дроссель PFC `L = 1 мГн` → ключ (MOSFET/IGBT) на землю → диод → выходной `C = 470 мкФ`.
2. Выходное напряжение задаём `U_вых = 400 В` (выше пика сети 311 В).
3. Нагрузка `R = 320 Ом` (≈ 500 Вт).

### B2. Управление (average current mode)
1. Внешний контур: ПИ-регулятор по `U_вых` (опора 400 В) задаёт амплитуду тока.
2. Опорный ток = выход регулятора × форма `|sin|` выпрямленного напряжения.
3. Внутренний контур тока: ПИ → ШИМ-компаратор (несущая 20–50 кГц).
4. Используйте готовый пример `power_PFC` из библиотеки Simscape как основу.

### B3. Измерения
1. Снимите `i_с(t)` — должна стать синусоидой в фазе с напряжением.
2. FFT тока: измерьте `THD` и `PF`.
