# Лекция 08. Коэффициент мощности и корректоры (PFC)

**Модуль II. Преобразователи AC/DC**

---

## 1. Полная, активная и реактивная мощность

В цепи переменного тока различают:
- **Активную мощность** `P` [Вт] — реально потребляемую (превращается в работу/тепло).
- **Полную мощность** `S = U_rms·I_rms` [ВА] — произведение действующих значений.
- **Коэффициент мощности** `PF = P/S` — какая доля полной мощности «полезна».

Для синусоидальных напряжения и тока `PF = cos φ`, где φ — сдвиг фаз. Но при **несинусоидальном** токе (что типично для выпрямителей) это уже неверно.

---

## 2. Коэффициент мощности при несинусоидальном токе

Если напряжение синусоидально, а ток содержит гармоники, активную мощность создаёт **только первая гармоника тока** (совпадающая по частоте с напряжением). Тогда:
```
P = U_rms·I₁·cos φ₁
S = U_rms·I_rms
PF = P/S = (I₁/I_rms)·cos φ₁ = k_иск·cos φ₁
```
где:
- `cos φ₁` — **коэффициент сдвига** (фазовый сдвиг первой гармоники),
- `k_иск = I₁/I_rms` — **коэффициент искажения** (доля первой гармоники в действующем токе).

**Важный вывод:** `PF = k_иск · cos φ₁`. Даже при `cos φ₁ = 1` (ток в фазе) высокое содержание гармоник (`k_иск < 1`) снижает коэффициент мощности.

---

## 3. THD и связь с коэффициентом искажения

**Суммарные гармонические искажения тока:**
```
THD = √(Σ_{n≥2} Iₙ²) / I₁
```
Действующий ток через гармоники: `I_rms = √(I₁² + Σ Iₙ²) = I₁·√(1 + THD²)`. Тогда:
```
k_иск = I₁/I_rms = 1/√(1 + THD²)
```
**Вывод:** связывает THD и коэффициент искажения. При `THD = 100%` (= 1.0): `k_иск = 1/√2 ≈ 0.707`.

---

## 4. Проблема диодного выпрямителя с C-фильтром

Как показано в лекции 7, выпрямитель с ёмкостным фильтром потребляет ток **короткими высокими импульсами** вблизи пиков напряжения. Такой ток:
- сильно несинусоидален (богат нечётными гармониками 3, 5, 7…),
- имеет `THD ≈ 80…130%`,
- даёт `PF ≈ 0.5…0.7`.

**Последствия для сети:** перегрузка нейтрали (3-я гармоника складывается), нагрев трансформаторов, искажение напряжения сети. Поэтому действуют нормы (**IEC/EN 61000-3-2**), ограничивающие гармоники тока, потребляемого из сети. Их выполнение требует коррекции коэффициента мощности — **PFC**.

---

## 5. Пассивная PFC

Простейший способ — последовательный **дроссель** (на входе или в звене постоянного тока). Он «размазывает» импульс тока во времени, снижая гармоники. 

**Плюсы:** простота, надёжность, нет помех. **Минусы:** большие габариты и масса дросселя, ограниченное улучшение (`PF` до ~0.9), нельзя стабилизировать выход. Применяется в недорогой технике средней мощности.

---

## 6. Активная PFC на повышающем преобразователе

### 6.1. Принцип

Между выпрямителем и нагрузкой ставят **Boost-преобразователь** (лекция 9), управляемый так, чтобы потребляемый из сети ток **повторял форму выпрямленного напряжения** (синусоиду). Тогда вход «выглядит» для сети как чисто активное сопротивление.

Система управления имеет два контура:
- **внешний** — стабилизирует выходное напряжение DC-шины;
- **внутренний (контур тока)** — заставляет входной ток следовать за формой `|sin|` напряжения сети (метод average current mode).

### 6.2. Почему именно Boost и почему выход выше пика сети

Boost способен работать при входном напряжении от нуля до пика и обеспечивает непрерывный входной ток (через дроссель), что удобно для формирования синусоиды. Выходное напряжение PFC выбирают **выше амплитуды сети** (например, 400 В при сети 230 В, пик ≈ 325 В), потому что Boost не может выдать напряжение ниже входного — иначе контроль тока теряется вблизи пиков.

**Описание графиков:** до PFC ток сети — узкие пики (форма «акульих плавников»); после активного PFC ток — гладкая синусоида, совпадающая по фазе с напряжением. Спектр: до — мощные 3-я, 5-я, 7-я гармоники; после — почти только первая.

---

## 7. Численный пример

**Выпрямитель + C-фильтр:** `THD ≈ 100%` → `k_иск ≈ 0.707`, `cos φ₁ ≈ 1`:
```
PF ≈ 0.707·1 ≈ 0.71
```
**С активным PFC:** `THD < 5%` → `k_иск ≈ 0.999`:
```
PF ≈ 0.99
```

---

## 8. Выводы

1. При несинусоидальном токе `PF = k_иск·cos φ₁`, а не просто `cos φ`.
2. `k_иск = 1/√(1+THD²)`; гармоники снижают коэффициент мощности даже без фазового сдвига.
3. Диодный выпрямитель с C-фильтром имеет высокий THD и низкий PF, нарушая нормы IEC 61000-3-2.
4. Активный PFC (Boost) формирует синусоидальный входной ток, поднимая PF до ~0.99; выход берут выше пика сети.
