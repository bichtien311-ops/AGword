# Лекция 14. Системы управления преобразователями

**Модуль IV. Управление, ЭМС и применение**

---

## 1. Зачем нужна обратная связь

В разомкнутой схеме скважность `D` задаётся вручную, и выходное напряжение «плавает» при изменении входного напряжения и тока нагрузки. Чтобы выход оставался стабильным, применяют **замкнутое управление**: измеряют выходную величину, сравнивают с заданием и корректируют `D` регулятором.

Цели управления:
- **точность** — малая статическая ошибка;
- **быстродействие** — быстрая отработка возмущений;
- **устойчивость** — отсутствие колебаний и запас по фазе/амплитуде.

---

## 2. Усреднённая модель преобразователя

Для анализа управления импульсный преобразователь заменяют **усреднённой по периоду** моделью (averaged model), а затем линеаризуют вокруг рабочей точки (**модель малого сигнала**, small-signal). Получают передаточную функцию «управление → выход» `G_vd(s) = û_вых(s)/d̂(s)`.

Для Buck в CCM это звено 2-го порядка (из-за LC-фильтра):
```
G_vd(s) = U_вх / (1 + s·L/R + s²·L·C)
```
с резонансом на `f_LC = 1/(2π√(LC))`. Boost и Buck-Boost дополнительно имеют **правый нуль (RHP zero)**, ограничивающий быстродействие: при росте D выход сначала кратковременно проседает.

---

## 3. ПИ-регулятор

Самый распространённый регулятор:
```
G_PI(s) = K_p + K_i/s
```
- **Пропорциональная часть `K_p`** — реагирует на текущую ошибку, влияет на быстродействие.
- **Интегральная часть `K_i`** — накапливает ошибку, **обнуляет статическую ошибку** (в установившемся режиме выход точно равен заданию).

Дифференциальную часть (ПИД) в силовых преобразователях применяют редко — она усиливает шум.

---

## 4. Контур напряжения и тока

### 4.1. Одноконтурное управление (voltage mode)
Один регулятор по напряжению формирует `D`. Просто, но реакция на возмущения медленнее, а резонанс LC требует тщательной коррекции.

### 4.2. Двухконтурное управление (current mode)
- **Внутренний контур тока** — быстро управляет током дросселя/ключа.
- **Внешний контур напряжения** — задаёт уставку тока.

Преимущества current-mode: встроенное ограничение тока (защита), упрощённая динамика (контур тока «убирает» один полюс), лучшая реакция на изменение входного напряжения.

---

## 5. Частотный анализ устойчивости

Качество замкнутой системы оценивают по **петлевому усилению** `T(s) = G_рег·G_vd·H` (H — коэффициент ОС).

По диаграмме Боде:
- **Частота среза `f_c`** — где `|T| = 1` (0 дБ); определяет быстродействие.
- **Запас по фазе `PM = 180° + arg[T(jω_c)]`** — должен быть ≥ 45° (лучше 60°).
- **Запас по амплитуде `GM`** — насколько `|T|` ниже 0 дБ на частоте, где фаза = −180°.

**Правила проектирования:**
- частоту среза берут `f_c ≤ f_sw/10` (усреднённая модель верна только ниже частоты коммутации);
- запас по фазе `PM ≈ 45…60°` — компромисс быстродействия и отсутствия перерегулирования.

**Описание диаграммы Боде:** ЛАЧХ петлевого усиления спадает с ростом частоты и пересекает 0 дБ на `f_c`; на этой частоте по ЛФЧХ отсчитывают расстояние фазы до −180° — это и есть запас по фазе. Регулятор (`K_p`, `K_i`) «двигает» кривую так, чтобы получить нужные `f_c` и `PM`.

---

## 6. Численный пример

Buck из ЛР-09, `f_sw = 100 кГц`. Цель: `f_c = f_sw/10 = 10 кГц`, `PM ≈ 60°`.
Подбирают `K_p` так, чтобы усиление пересекало 0 дБ на 10 кГц, и `K_i` так, чтобы интегральное действие подняло усиление на низких частотах (точность), не «съев» запас фазы у `f_c`.

---

## 7. Выводы

1. Обратная связь стабилизирует выход при изменениях входа и нагрузки.
2. Анализ ведут по усреднённой малосигнальной модели; у Boost/Buck-Boost есть RHP-нуль, ограничивающий быстродействие.
3. ПИ-регулятор: `K_p` — быстродействие, `K_i` — нулевая статическая ошибка.
4. Устойчивость оценивают по `f_c` (≤ f_sw/10) и запасу по фазе (≥ 45°); current-mode даёт защиту по току и лучшую динамику.
