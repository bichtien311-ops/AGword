# Лекция 16. Применение: электропривод, ИВЭП, возобновляемая энергетика

**Модуль IV. Управление, ЭМС и применение**

---

## 1. Цель лекции

Связать изученные топологии (выпрямители, DC/DC, инверторы, управление) с реальными системами. Силовая электроника редко применяется «сама по себе» — это всегда часть энергетической системы со своими требованиями.

---

## 2. Частотно-регулируемый электропривод

### 2.1. Структура

Самое массовое применение силовой электроники. Типовая схема преобразователя частоты:
```
Сеть AC → Выпрямитель → DC-шина (C) → Трёхфазный инвертор → Двигатель
```
- Выпрямитель (часто неуправляемый диодный) создаёт постоянное напряжение.
- **DC-шина** с конденсатором сглаживает и развязывает выпрямитель и инвертор: они работают независимо.
- Инвертор (SPWM/SVPWM, лекция 12) формирует трёхфазное напряжение **регулируемой частоты и амплитуды** для двигателя.

### 2.2. Закон U/f = const

Скорость асинхронного двигателя пропорциональна частоте. Чтобы магнитный поток оставался постоянным (а значит, момент — номинальным), напряжение меняют пропорционально частоте:
```
U/f = const
```
**Обоснование:** поток `Φ ∝ U/f` (из закона электромагнитной индукции). При снижении частоты без снижения напряжения поток вырос бы и насытил сердечник. Поэтому U и f регулируют совместно.

При очень низких частотах добавляют **компенсацию IR-падения** (boost), иначе момент падает.

### 2.3. Векторное управление (обзор)
Для точного управления моментом и скоростью применяют **полеориентированное управление (FOC)**: ток статора разделяется на составляющую потока и составляющую момента, которыми управляют раздельно (как в машине постоянного тока). Это даёт высокое быстродействие, но требует датчиков и вычислений (DSP).

### 2.4. Преимущество частотного пуска
Прямой пуск асинхронного двигателя даёт пусковой ток в 5–7 раз больше номинального. Частотный пуск (плавный разгон частоты от нуля) ограничивает пусковой ток до номинального уровня, снижает механические удары и экономит энергию.

## 3. Импульсные источники питания (ИВЭП)

Современный сетевой блок питания обычно состоит из каскадов:
```
Сеть → EMI-фильтр → Выпрямитель → PFC (Boost) → Изолированный DC/DC (LLC/Flyback) → Выход
```
- **EMI-фильтр** (лекция 15) — подавление помех.
- **PFC** (лекция 8) — высокий коэффициент мощности, шина ~400 В.
- **Изолированный DC/DC** (лекции 10, 13) — развязка и формирование выходных напряжений.
- **Дежурный источник (standby)** — маломощный Flyback для питания управления.

Тенденция: высокая частота (SiC/GaN), резонансные топологии (LLC) → высокий КПД и плотность мощности.

## 4. Возобновляемая энергетика

### 4.1. Солнечная (PV)
Солнечная панель имеет нелинейную ВАХ с **точкой максимальной мощности (MPP)**, зависящей от освещённости и температуры. Преобразователь (обычно Boost) должен удерживать рабочую точку в MPP — алгоритм **MPPT**.

Метод **P&O (perturb & observe):** периодически изменяют рабочее напряжение и наблюдают мощность; если она выросла — продолжают в том же направлении, иначе разворачиваются. Условие максимума:
```
dP/dV = 0
```
Затем сетевой инвертор отдаёт мощность в сеть.

### 4.2. Ветровая
Генератор переменной частоты → выпрямитель → DC-шина → сетевой инвертор (схема AC/DC/AC). Это развязывает частоту вращения турбины от частоты сети.

### 4.3. Накопители энергии
Аккумуляторы подключают через **двунаправленный DC/DC** (заряд/разряд) и инвертор, способный передавать мощность в обе стороны.

---

## 5. Сетевые инверторы

Инвертор, работающий **параллельно с сетью**, должен синхронизироваться с ней по частоте и фазе. Для этого применяют **ФАПЧ (PLL)** — систему слежения за фазой сетевого напряжения.

Передаваемая активная мощность зависит от угла δ между напряжениями инвертора и сети:
```
P = (U_инв·U_сети / X)·sin δ
```
где X — реактивное сопротивление связи. Регулируя δ (активная мощность) и амплитуду (реактивная), инвертор управляет потоком энергии в сеть.

---

## 6. Численный пример

**Привод:** двигатель 50 Гц/380 В. Для работы на 25 Гц при U/f = const:
```
U = 380·(25/50) = 190 В
```
поток сохраняется, момент остаётся номинальным.

**PV + Boost:** панель `U_mpp = 30 В`, нужна шина 400 В:
```
D = 1 − U_вх/U_вых = 1 − 30/400 ≈ 0.925
```
Такая высокая скважность непрактична (большие потери) — поэтому используют каскадирование или изолированный преобразователь с трансформатором.

---

## 7. Выводы

1. Частотный привод: выпрямитель → DC-шина → инвертор; регулирование по закону U/f = const сохраняет поток и момент.
2. DC-шина развязывает входной и выходной каскады, позволяя им работать независимо.
3. ИВЭП — каскад EMI-фильтр + PFC + изолированный DC/DC; тренд на LLC и SiC/GaN.
4. В ВИЭ ключевые задачи — MPPT (PV), AC/DC/AC (ветро), двунаправленный DC/DC (накопители) и синхронизация с сетью (PLL).
