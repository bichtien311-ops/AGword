# Лабораторная 16. Преобразователь для PV / электропривода

**Тема:** Прикладная система — MPPT для PV или частотный пуск АД по закону U/f.

## Цель
Применить изученные топологии: реализовать MPPT для солнечной панели (вариант A) ИЛИ частотный пуск асинхронного двигателя (вариант B).

## Инструменты
MATLAB/Simulink (Simscape Electrical: PV Array, Asynchronous Machine, Universal Bridge).

---

## Вариант A — PV + MPPT (Boost)

### A1. Теория
Панель имеет точку максимальной мощности (MPP). Алгоритм P&O: меняем рабочее напряжение, наблюдаем `P`; растёт — продолжаем, падает — разворот. Условие максимума `dP/dV = 0`.

### A2. Схема
1. `PV Array`: например 1 строка, 60-элементная панель, `P_max ≈ 250 Вт`, `U_mpp ≈ 30 В`.
2. `Boost` (из ЛР-09): `L = 1 мГн`, `C = 470 мкФ`, выход на нагрузку/DC-шину.
3. MPPT-контроллер (`MATLAB Function` с P&O): вход `U_pv`, `I_pv`; выход — скважность D.

### A3. Эксперимент
1. Решатель `ode23tb`, Stop time `2 с`.
2. В t = 1 с измените освещённость (Irradiance) 1000 → 600 Вт/м².
3. Снимите `P_pv(t)` — MPPT должен вернуть рабочую точку к новому максимуму.
4. Сравните отдаваемую мощность с MPPT и без (фиксированный D).

---

## Вариант B — Электропривод U/f

### B1. Теория
Закон `U/f = const` сохраняет магнитный поток. При `f < f_ном` напряжение снижают пропорционально.

### B2. Схема
1. `DC Voltage Source = 540 В` → `Universal Bridge` (3 arms, IGBT) → `Asynchronous Machine` (например 3 кВт, 380 В, 50 Гц).
2. Управление: SPWM, частота-задание `f` нарастает 0 → 50 Гц за 1 с (рампа), амплитуда `m_a ∝ f` (U/f).
3. Нагрузка на валу — момент сопротивления (`Tm`).

### B3. Эксперимент
1. Решатель `ode23tb`, Stop time `2 с`.
2. Снимите скорость ротора, ток статора, электромагнитный момент при пуске.
3. Сравните пусковой ток с прямым пуском (подача 50 Гц сразу).
