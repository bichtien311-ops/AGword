# Лекция 07. Сглаживающие фильтры

**Модуль II. Преобразователи AC/DC**

---

## 1. Зачем нужен фильтр

Выпрямленное напряжение содержит постоянную составляющую `U_ср` и переменную (пульсации). Большинству нагрузок (электроника, DC-шина) нужно стабильное напряжение, поэтому между выпрямителем и нагрузкой ставят **сглаживающий фильтр** — он подавляет пульсации, пропуская постоянную составляющую.

Эффективность фильтра оценивают **коэффициентом сглаживания**:
```
q = k_п(вход) / k_п(выход)
```
Чем больше `q`, тем лучше фильтр.

---

## 2. Ёмкостный фильтр (C)

### 2.1. Принцип

Конденсатор включается параллельно нагрузке. Он заряжается до пика выпрямленного напряжения, а когда напряжение спадает — разряжается на нагрузку, поддерживая её питание. Диоды проводят лишь короткими импульсами вблизи пиков.

### 2.2. Вывод пульсаций

В интервале между пиками (длительность ≈ `T_п = 1/f_пульс`) конденсатор отдаёт нагрузке заряд `Q = I_н·T_п`. Изменение напряжения:
```
ΔU = Q/C = I_н/(f_пульс·C)
```
Это размах пульсаций (приближённо, при `ΔU << U_ср`). Коэффициент пульсаций:
```
k_п = ΔU/(2·U_ср) = I_н/(2·f_пульс·C·U_ср)
```

**Вывод:** пульсации обратно пропорциональны `C` и частоте пульсаций. Поэтому трёхфазные мосты (f_пульс = 6f) требуют меньшей ёмкости, чем однофазные (2f).

### 2.3. Недостаток
Пиковые токи диодов растут с увеличением C (заряд за всё более короткое время). Это нагружает диоды и сеть.

---

## 3. Индуктивный фильтр (L)

### 3.1. Принцип

Дроссель включается **последовательно** с нагрузкой. Он препятствует изменению тока, сглаживая его. Эффективен при больших токах (мощные выпрямители) и при индуктивно-резистивной нагрузке.

### 3.2. Условие непрерывного тока

L-фильтр работает хорошо только если ток не прерывается. Существует **граничная (критическая) индуктивность `L_кр`** — ниже неё ток становится прерывистым, и фильтрация ухудшается. Для однофазного мостового выпрямителя порядок:
```
L_кр ≈ R/(3ω)
```
(зависит от схемы и числа пульсаций).

### 3.3. Особенность
L-фильтр **сглаживает ток**, тогда как C-фильтр **сглаживает напряжение**. L уменьшает пиковые токи диодов — в отличие от C.

---

## 4. LC-фильтр (Г-образный)

### 4.1. Принцип

Последовательный дроссель + параллельный конденсатор. Дроссель сглаживает ток, конденсатор — напряжение. Это фильтр нижних частот 2-го порядка.

### 4.2. Подавление пульсаций

Для гармоники пульсации с частотой `ω_п` коэффициент передачи переменной составляющей (при `ω_п >> ω_резонанса`):
```
|K| ≈ 1/(ω_п²·L·C − 1) ≈ 1/(ω_п²·L·C)
```
**Вывод:** подавление растёт как квадрат частоты пульсаций и произведение `L·C`. Резонансная частота фильтра `f_0 = 1/(2π√(LC))` должна быть **значительно ниже** частоты пульсаций, иначе возможен резонансный выброс.

### 4.3. Преимущество
LC даёт хорошее сглаживание при умеренных значениях L и C и снижает пиковые токи диодов (в отличие от чистого C). Это лучший компромисс для большинства задач.

---

## 5. П-образный фильтр (CLC)

Конденсатор + дроссель + конденсатор. Первый конденсатор уже снижает пульсации, затем LC дополнительно сглаживает. Даёт максимальное сглаживание, но первый C сохраняет проблему пиковых токов диодов. Применяется там, где нужны очень малые пульсации.

---

## 6. Сравнение фильтров

| Фильтр | Что сглаживает | Пиковый ток диодов | Габариты | Применение |
|--------|----------------|--------------------|----------|------------|
| C | напряжение | высокий | малые | маломощные ИВЭП |
| L | ток | низкий | большие (дроссель) | мощные, RL-нагрузка |
| LC | и то, и другое | умеренный | средние | универсальный компромисс |
| CLC | максимально | высокий | средние | малые пульсации |

**Описание графика:** на осциллограмме `U_н(t)` без фильтра — глубокие пульсации; с C — пилообразные колебания вокруг `U_ср`; с LC — почти прямая линия с малой рябью. Параллельно ток диода: без C — плавная полусинусоида, с C — узкие высокие импульсы.

---

## 7. Численный пример (C-фильтр)

Однофазный мост, `I_н = 1 А`, `f_пульс = 100 Гц`, `C = 1000 мкФ`, `U_ср ≈ 300 В`.
```
ΔU = I_н/(f_пульс·C) = 1/(100·1000·10⁻⁶) = 10 В (размах)
k_п = 5/300 ≈ 1.7%
```

---

## 8. Выводы

1. Фильтр подавляет переменную составляющую, пропуская постоянную; качество — коэффициент сглаживания `q`.
2. C-фильтр сглаживает напряжение, но создаёт пиковые токи диодов; пульсации `∝ 1/(f·C)`.
3. L-фильтр сглаживает ток, требует непрерывного режима (`L > L_кр`).
4. LC — оптимальный компромисс; подавление `∝ ω_п²·L·C`. Резонанс фильтра держат ниже частоты пульсаций.
