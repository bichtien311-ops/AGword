# Лабораторная 07. Сравнение сглаживающих фильтров

**Тема:** Исследование C-, L-, LC- и CLC-фильтров на выходе выпрямителя.

## Цель
Сравнить эффективность фильтров по коэффициенту пульсаций и оценить влияние на токи диодов.

## Теория и расчёт (до симуляции)
C-фильтр: `ΔU ≈ I_н/(f_пульс·C)`, `k_п = ΔU/(2·U_ср)`.
Дано: 1ф мост, `U_ср ≈ 198 В`, `R = 200 Ом` → `I_н ≈ 1 А`, `f_пульс = 100 Гц`.
Для `C = 470 мкФ`:
```
ΔU ≈ 1/(100·470·10⁻⁶) ≈ 21 В → k_п ≈ 10.5/198 ≈ 5.3%
```

---

## Базовая схема (LTspice)
1. Источник `SINE(0 311 50)`, мост из 4 диодов `1N4007`, нагрузка `R = 200 Ом`.
2. `.tran 0 200m 100m 10u`.
3. Измерьте `k_п` без фильтра (база ≈ 48%).

---

## Часть A. C-фильтр
1. Добавьте `C = 470 мкФ` параллельно R.
2. Измерьте `k_п` (≈ 5%) и пиковый ток диода `I(D1)`.

## Часть B. L-фильтр
1. Уберите C. Включите дроссель `L = 1 Гн` последовательно с R.
2. Измерьте `k_п`, отметьте сглаживание формы тока (импульсы диода исчезают).

## Часть C. LC-фильтр
1. Дроссель `L = 0.5 Гн` последовательно + `C = 220 мкФ` параллельно R.
2. Проверьте резонансную частоту: `f_0 = 1/(2π√(LC)) ≈ 1/(2π√(0.5·220e-6)) ≈ 15 Гц` (много ниже 100 Гц — хорошо).
3. Измерьте `k_п` (должен быть < 1%).

## Часть D. CLC (П-фильтр)
1. `C1 = 220 мкФ` + `L = 0.5 Гн` + `C2 = 220 мкФ`.
2. Измерьте `k_п` (минимальный).

2. Почему важно держать `f_0` фильтра ниже частоты пульсаций?

## Форма отчёта
