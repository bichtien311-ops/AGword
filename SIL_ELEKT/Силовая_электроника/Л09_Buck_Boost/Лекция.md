# Лекция 09. Понижающий (Buck) и повышающий (Boost) преобразователи

**Модуль III. Преобразователи DC/DC и DC/AC**

---

## 1. Идея импульсного DC/DC-преобразования

Чтобы изменить уровень постоянного напряжения с высоким КПД, его «нарезают» ключом и сглаживают LC-фильтром. Уровень выхода задаётся **скважностью (коэффициентом заполнения)**:
```
D = t_вкл / T,   0 ≤ D ≤ 1
```
где `T = 1/f_sw` — период коммутации. Управление скважностью при постоянной частоте — это **широтно-импульсная модуляция (ШИМ, PWM)**.

Ключевой метод анализа — **баланс вольт-секунд на дросселе**: в установившемся режиме среднее напряжение на дросселе за период равно нулю (иначе ток рос бы бесконечно). Это даёт передаточные функции всех топологий.

---

## 2. Понижающий преобразователь (Buck)

### 2.1. Схема и фазы работы

Ключ — последовательно с входом; после него — узел, к которому подключены диод (или синхронный ключ) на землю и дроссель L к выходному конденсатору C и нагрузке.

- **Фаза 1 (ключ замкнут, `t_вкл = D·T`):** напряжение на дросселе `U_L = U_вх − U_вых`, ток дросселя нарастает.
- **Фаза 2 (ключ разомкнут, `t_выкл = (1−D)·T`):** ток дросселя замыкается через диод, `U_L = −U_вых`, ток спадает.

### 2.2. Вывод передаточной функции (баланс вольт-секунд)

Среднее напряжение на L за период = 0:
```
(U_вх − U_вых)·D·T + (−U_вых)·(1−D)·T = 0
U_вх·D − U_вых·D − U_вых + U_вых·D = 0
U_вх·D − U_вых = 0
```
Отсюда:
```
U_вых = D·U_вх
```
Так как `0 ≤ D ≤ 1`, выход **всегда ниже или равен** входу — отсюда название.

### 2.3. Пульсации тока дросселя

В фазе 1: `ΔI_L = (U_L/L)·t_вкл = (U_вх − U_вых)·D·T / L`. Подставляя `U_вых = D·U_вх`:
```
ΔI_L = U_вх·D·(1 − D)/(L·f_sw)
```
Максимальна при `D = 0.5`.

### 2.4. Пульсации выходного напряжения

Переменная составляющая тока дросселя (треугольник `ΔI_L`) заряжает/разряжает C. Интегрируя треугольный ток:
```
ΔU_C = ΔI_L/(8·f_sw·C)
```
**Вывод:** заряд за половину периода `ΔQ = (1/2)·(T/2)·(ΔI_L/2) = ΔI_L·T/8`; `ΔU_C = ΔQ/C = ΔI_L/(8·f·C)`.

## 3. Повышающий преобразователь (Boost)

### 3.1. Схема и фазы работы

Дроссель L — последовательно со входом; далее ключ на землю и диод к выходному конденсатору C/нагрузке.

- **Фаза 1 (ключ замкнут):** дроссель подключён к входу, `U_L = U_вх`, ток нарастает, энергия запасается в L. Диод закрыт; нагрузку питает C.
- **Фаза 2 (ключ разомкнут):** ток дросселя течёт через диод в нагрузку, `U_L = U_вх − U_вых` (отрицательно, ток спадает). Запасённая энергия добавляется к входной.

### 3.2. Вывод передаточной функции

Баланс вольт-секунд:
```
U_вх·D·T + (U_вх − U_вых)·(1−D)·T = 0
U_вх·D + U_вх·(1−D) − U_вых·(1−D) = 0
U_вх − U_вых·(1−D) = 0
```
Отсюда:
```
U_вых = U_вх/(1 − D)
```
Так как `(1−D) ≤ 1`, выход **всегда выше или равен** входу. При `D → 1` теоретически `U_вых → ∞`, но реально ограничено потерями (КПД падает, кривая загибается вниз).

### 3.3. Пульсации тока дросселя
```
ΔI_L = U_вх·D/(L·f_sw)
```

---

## 4. Режимы непрерывного (CCM) и прерывистого (DCM) тока

- **CCM (continuous conduction mode):** ток дросселя не спадает до нуля. Передаточные функции выше справедливы.
- **DCM (discontinuous):** при малой нагрузке ток успевает упасть до нуля и «лежит» на нуле часть периода. Передаточная функция меняется — становится зависимой и от нагрузки, что усложняет управление.

**Граница режимов** определяется критической индуктивностью. Для Buck:
```
L_кр = (1 − D)·R/(2·f_sw)
```
При `L > L_кр` — режим CCM при данной нагрузке `R`.

**Описание графика `i_L(t)`:** в CCM — треугольная пила вокруг среднего тока (ток нагрузки), не касающаяся нуля; в DCM — треугольники, начинающиеся от нуля, с интервалом нулевого тока между ними.

---

## 5. КПД и потери

Основные потери: проводимость ключа и диода, переключение, активное сопротивление дросселя (`R_L`) и ESR конденсатора. Синхронный Buck (диод заменён на MOSFET) повышает КПД, особенно при низких выходных напряжениях.

---

## 6. Численный пример (Buck)

`U_вх = 12 В`, `D = 0.5`, `f_sw = 100 кГц`, `L = 47 мкГн`, `C = 100 мкФ`, `R = 5 Ом`.
- `U_вых = 0.5·12 = 6 В`, `I_вых = 6/5 = 1.2 А`.
- `ΔI_L = 12·0.5·0.5/(47·10⁻⁶·100·10³) = 3/(4.7) ≈ 0.64 А`.
- `ΔU_C = 0.64/(8·100·10³·100·10⁻⁶) = 0.64/80 ≈ 8 мВ`.

---

## 7. Выводы

1. ШИМ задаёт выход через скважность D; анализ — через баланс вольт-секунд на дросселе.
2. Buck: `U_вых = D·U_вх` (понижает); Boost: `U_вых = U_вх/(1−D)` (повышает).
3. Пульсации тока дросселя `∝ 1/(L·f_sw)`; пульсации напряжения `∝ 1/(C·f_sw)`.
4. CCM/DCM разграничены критической индуктивностью; в DCM передаточная функция зависит от нагрузки.
