# Лабораторная 11. Однофазный мостовой инвертор (SPWM)

**Тема:** H-мост с синусоидальной ШИМ, спектр выходного напряжения.

## Цель
Сформировать синусоидальное выходное напряжение методом SPWM, исследовать влияние m_a и m_f, оценить THD до и после фильтра.

## Теория и расчёт (до симуляции)
В линейной зоне (мост): `U₁m = m_a·U_dc`.
Дано: `U_dc = 320 В`, `f₁ = 50 Гц`, несущая `f_нес = 50·21 = 1050 Гц` (m_f = 21).
- `m_a = 0.8` → `U₁m = 0.8·320 = 256 В`, `U₁(rms) ≈ 181 В`.

---

## Часть A. Схема (Simulink)

### A1. Силовая часть
1. `DC Voltage Source = 320 В`.
2. `Universal Bridge`: arms = 2, device = `IGBT/Diodes` (H-мост, 4 ключа).
3. Нагрузка: `R = 20 Ом` + `L = 20 мГн` (RL).

### A2. SPWM-генератор
1. `Sine Wave`: amplitude = m_a, freq = 2π·50 рад/с.
2. `Repeating Sequence` (треугольник) amplitude ±1, freq = 1050 Гц — несущая.
3. `Relational Operator` (>=) сравнивает синус с несущей → сигналы на ключи (с инверсией для второй диагонали).
4. Для униполярной модуляции — второй компаратор с инвертированным синусом.

### A3. Решатель
- `ode23tb`, Max step `1e-6`, Stop time `0.1`.

---

## Часть B. Измерения

### B1. Спектр до фильтра
1. Снимите `U_вых(t)` (до фильтра) — ШИМ-импульсы.
2. `powergui → FFT Analysis`: измерьте `U₁` (≈181 В RMS при m_a=0.8) и `THD`.
3. Убедитесь, что гармоники сгруппированы около m_f·f₁ = 1050 Гц.

### B2. Влияние m_a
Снимите `U₁` при `m_a = 0.4, 0.8, 1.0, 1.2`. Постройте `U₁(m_a)`, отметьте перемодуляцию при `m_a > 1` (рост U₁ замедляется, появляются НЧ-гармоники).

### B3. LC-фильтр
1. Добавьте выходной `L_ф = 2 мГн` + `C_ф = 10 мкФ` (f_срез ≈ 1.1 кГц).
2. Снимите `U_вых` после фильтра — синусоида. Измерьте `THD` (резко падает).

### B4. Униполярная vs биполярная
Сравните THD для двух типов модуляции.
