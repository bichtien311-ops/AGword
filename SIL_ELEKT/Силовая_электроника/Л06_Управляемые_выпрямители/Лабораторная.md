# Лабораторная 06. Управляемый выпрямитель и регулировочная характеристика

**Тема:** Тиристорный управляемый мост, зависимость U_ср(α), режимы R и RL.

## Цель
Исследовать фазовое регулирование, построить регулировочную характеристику и сравнить R- и RL-нагрузку, наблюдать инверторный режим.

## Теория и расчёт (до симуляции)
- R-нагрузка: `U_ср = (U_m/π)·(1 + cos α)` (полностью управляемый мост).
- RL-нагрузка (непрерывный ток): `U_ср = (2·U_m/π)·cos α`.

`U_m = 311 В` (220 В RMS). Предрасчёт для RL:
| α | U_ср = (2·311/π)·cos α |
|----|------------------------|
| 0° | 198 В |
| 30° | 171 В |
| 60° | 99 В |
| 90° | 0 В |
| 120°| −99 В (инверторный) |

## Часть A. Схема (Simulink)

### A1. Сборка
1. `AC Voltage Source`: 311 В амплитуда, 50 Гц.
2. `Universal Bridge`: arms = 2, device = `Thyristors` (однофазный мост).
3. Нагрузка: сначала `R = 50 Ом`.
4. Управление: `Synchronized 6-Pulse Generator` или для 1ф — два `Pulse Generator` с задержкой по α, синхронизированные с сетью.
5. `powergui` (Continuous), Scope, Mean (окно 0.02 с).

### A2. Задание угла α
Угол α задаётся задержкой импульса: `t_α = α°/360 · 0.02 с`.
| α | задержка, мс |
|----|--------------|
| 30° | 1.67 |
| 60° | 3.33 |
| 90° | 5.00 |
| 120°| 6.67 |

### A3. Настройка решателя
- `ode23tb`, Variable-step, Max step `1e-5`, Stop time `0.2`.
