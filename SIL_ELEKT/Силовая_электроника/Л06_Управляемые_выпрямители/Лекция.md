# Лекция 06. Управляемые выпрямители (тиристорные)

**Модуль II. Преобразователи AC/DC**

---

## 1. Зачем управляемый выпрямитель

Если заменить диоды тиристорами, можно **регулировать** выходное напряжение, задерживая момент их включения. Это нужно для регулируемых источников питания, заряда аккумуляторов, управляемого электропривода постоянного тока и рекуперации энергии в сеть.

Управление осуществляется **углом α** (угол управления/открытия) — задержкой отпирающего импульса относительно момента естественного открытия диода (точки, где напряжение становится положительным для данного вентиля).

---

## 2. Однофазный управляемый мост

### 2.1. Активная нагрузка (R)

Тиристоры открываются при `ωt = α` и проводят до конца полупериода (`ωt = π`), когда ток спадает до нуля. Среднее напряжение (используются оба полупериода):
```
U_ср = (1/π) ∫_α^π U_m·sin θ dθ = (U_m/π)·(1 + cos α)
```
**Анализ:** `α = 0` → `U_ср = 2U_m/π` (как неуправляемый мост); `α = π` → `U_ср = 0`. Регулировочная характеристика — спадающая выпуклая кривая.

### 2.2. Активно-индуктивная нагрузка (RL, режим непрерывного тока)

При большой индуктивности ток не успевает спасть до нуля — тиристоры проводят и в области отрицательного напряжения (энергия индуктивности поддерживает ток). Среднее напряжение:
```
U_ср = (1/π) ∫_α^{π+α} U_m·sin θ dθ = (2·U_m/π)·cos α
```
**Вывод:** интегрирование на интервале длиной `π`, начинающемся в `α`:
`∫_α^{π+α} sin θ dθ = [−cos θ]_α^{π+α} = −cos(π+α) + cos α = cos α + cos α = 2cos α`. Отсюда `U_ср = (U_m/π)·2cos α`.

**Важное следствие:** при `α = 90°` → `U_ср = 0`; при `α > 90°` → `U_ср < 0` — **инверторный режим**.

---

## 3. Трёхфазный управляемый мост

Аналогично неуправляемому (лекция 5), но с задержкой α:
```
U_ср = (3√3·U_фm/π)·cos α = U_ср0·cos α
```
где `U_ср0 = 3√3·U_фm/π` — максимум (при α = 0). Зависимость `cos α` означает: характеристика симметрична, переход через ноль при α = 90°.

---

## 4. Инверторный режим

При `α > 90°` среднее напряжение становится отрицательным. Если в цепи есть источник ЭДС (например, машина постоянного тока в режиме генератора или заряженный аккумулятор), ток продолжает течь в прежнем направлении, но **мощность теперь передаётся из DC-цепи в сеть** — выпрямитель работает инвертором, ведомым сетью.

Вводят **угол опережения инвертора `β = 180° − α`**. Чтобы тиристоры успели восстановить запирающие свойства, `β` ограничивают снизу (`β > β_min`), иначе произойдёт «опрокидывание инвертора» — авария.

**Описание графика `U_ср(α)`:** монотонно спадающая кривая `∝ cos α`: положительная при α < 90° (выпрямление), ноль при 90°, отрицательная при α > 90° (инвертирование). Для R-нагрузки кривая идёт по `(1 + cos α)` и не уходит в минус.

---

## 5. Коммутация и угол μ

В реальной сети есть индуктивность (рассеяния трансформатора, линии). Из-за неё ток не может мгновенно переключиться с одного вентиля на другой — возникает **интервал коммутации `μ`**, в течение которого проводят оба тиристора. Это:
- немного снижает реальное `U_ср` (коммутационное падение, `∝ μ`);
- создаёт «провалы» (notches) в сетевом напряжении.

---

## 6. Влияние на сеть

Управляемый выпрямитель потребляет из сети ток, **сдвинутый по фазе** относительно напряжения (сдвиг растёт с α). Это означает потребление **реактивной мощности**:
```
Q ∝ sin α
```
Кроме того, форма тока несинусоидальна (содержит гармоники). Поэтому коэффициент мощности тиристорных выпрямителей падает с ростом α — серьёзный недостаток, отчасти решаемый PFC (лекция 8) или активными выпрямителями.

---

## 7. Численный пример

Трёхфазный мост, `U_ср0 = 513 В`:
- `α = 45°`: `U_ср = 513·cos 45° ≈ 363 В`.
- `α = 120°`: `U_ср = 513·cos 120° = −256 В` (инверторный режим — отдача в сеть).

---

## 8. Выводы

1. Угол управления α задаёт среднее напряжение; для RL-нагрузки `U_ср = (2U_m/π)·cos α`.
2. При α > 90° возможен инверторный режим — возврат энергии в сеть.
3. Индуктивность сети вызывает угол коммутации μ, снижающий `U_ср`.
4. Тиристорный выпрямитель потребляет реактивную мощность и гармоники — ухудшает коэффициент мощности.
