# Лабораторная 02. ВАХ диода/тиристора и фазовое управление

**Тема:** Характеристики силового диода и тиристора, фазовое регулирование.

## Цель
Снять прямую ВАХ диода и тиристора, исследовать включение тиристора управляющим импульсом и зависимость `U_ср(α)`.

## Теория и расчёт (до симуляции)
Однополупериодная схема с тиристором, R-нагрузка:
```
U_ср = (U_m/2π)·(1 + cos α)
```
Сеть `U_rms = 220 В` → `U_m = 311 В`. Предрасчёт:
| α | U_ср = (311/2π)(1+cos α) |
|----|--------------------------|
| 30° | 49.5·(1.866) ≈ 92.4 В |
| 60° | 49.5·(1.5) ≈ 74.2 В |
| 90° | 49.5·(1.0) ≈ 49.5 В |
| 120°| 49.5·(0.5) ≈ 24.7 В |

## Часть A. ВАХ диода (LTspice)

### A1. Схема
1. Источник V1 → диод D1 (например `1N4007` или `Dbreak`) → земля.
2. ПКМ по диоду → выберите модель из библиотеки.

### A2. Анализ DC-развёртки
1. Директива: `.dc V1 0 1 0.01` (от 0 до 1 В, шаг 10 мВ).
2. Run → постройте `I(D1)` от `V(вход)`.
3. Определите `U_F` при номинальном токе (например, при 1 А).

---

## Часть B. Фазовое управление тиристором (Simulink — рекомендуется)

### B1. Сборка схемы
1. Блоки (Simscape Electrical / Specialized Power Systems):
   - `AC Voltage Source`: 220 В (RMS) → амплитуда 311 В, частота 50 Гц;
   - `Thyristor`;
   - `Series RLC Branch` как R = 50 Ом (активная нагрузка);
   - `Pulse Generator` → на управляющий электрод (g);
   - `powergui` (Continuous), `Scope`, `Mean` (Mean Value, окно 0.02 с).
2. Соедините: источник → тиристор (A–K) → R → обратно к источнику.

### B2. Задание угла α через задержку импульса
Угол α в радианах соответствует времени `t_α = α/(2π·f) = α°/360 · 0.02 с`.
| α | Задержка импульса (Phase delay), с |
|----|--------------------------------------|
| 30° | 0.00167 |
| 60° | 0.00333 |
| 90° | 0.00500 |
| 120°| 0.00667 |
Настройте `Pulse Generator`: Period = 0.02 с, Pulse Width = 5%, Phase delay = из таблицы.

### B3. Настройка решателя
1. Solver: `ode23tb`, Variable-step, Max step `1e-5`.
2. Stop time: `0.1` (5 периодов).

### B4. Измерения
1. Для каждого α (30°, 60°, 90°, 120°) запустите модель.
2. Снимите осциллограмму `U_н(t)`, зафиксируйте `Mean` (U_ср) на установившемся периоде.
3. Занесите в таблицу, сравните с предрасчётом.
4. Постройте регулировочную характеристику `U_ср(α)` (например, через `plot` в MATLAB по снятым точкам).
