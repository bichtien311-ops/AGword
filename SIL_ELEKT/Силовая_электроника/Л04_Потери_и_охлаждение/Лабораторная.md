# Лабораторная 04. Расчёт потерь и теплового режима

**Тема:** Бюджет потерь транзистора, выбор радиатора, проверка снаббера.

## Цель
Связать электрические потери с тепловым режимом: рассчитать `P_потерь`, подобрать радиатор и проверить тепловую модель в симуляции.

## Теория и расчёт (до симуляции)
```
T_j = P·(R_th(jc) + R_th(cs) + R_th(sa)) + T_a
R_th(sa) ≤ (T_j(max) − T_a)/P − R_th(jc) − R_th(cs)
```
Исходные данные:
- `P_total = 20 Вт`, `T_j(max) = 125 °C`, `T_a = 40 °C`;
- `R_th(jc) = 0.5 °C/Вт`, `R_th(cs) = 0.2 °C/Вт`.

Расчёт допустимого радиатора:
```
R_th(sa) ≤ (125 − 40)/20 − 0.5 − 0.2 = 4.25 − 0.7 = 3.55 °C/Вт
```

---

## Часть A. Бюджет потерь (из модели ЛР-03)
1. Возьмите модель Buck/переключения из ЛР-03 в рабочей точке (`U`, `I`, `f_sw = 50 кГц`).
2. Измерьте средние потери проводимости: `⟨Id²⟩·R_DS(on)` (или `⟨V_DS·Id⟩` в открытом состоянии).
3. Измерьте потери переключения `P_sw = (E_on+E_off)·f_sw`.
4. Сложите: `P_total = P_cond + P_sw`.

---

## Часть B. Тепловая модель (электрическая аналогия, LTspice)

### B1. Сборка RC-тепловой цепочки
Аналогия: мощность → ток, температура → напряжение, `R_th` → сопротивление, `C_th` → ёмкость.
1. Источник тока `I1 = P_total` (например 20 А = 20 Вт).
2. Последовательно резисторы: `R_jc = 0.5`, `R_cs = 0.2`, `R_sa = 3.55` (Ом = °C/Вт).
3. Параллельно узлам — конденсаторы теплоёмкости (например `C_j = 1m`, `C_s = 0.1`).
4. Нижний узел через источник напряжения `V_amb = 40 В` (= 40 °C) к земле.

### B2. Анализ
1. `.tran 0 60 0 0.01` (60 с — выход на тепловой режим).
2. Напряжение на узле кристалла = `T_j`. Проверьте установившееся значение:
```
T_j = 20·(0.5+0.2+3.55) + 40 = 20·4.25 + 40 = 125 °C ✓
```
3. Снимите переходную кривую `T_j(t)` — экспоненциальный выход на режим.

### B3. Проверка границы
Уменьшите `R_sa` до 2.0 — `T_j` снизится; увеличьте до 5.0 — `T_j` превысит 125 °C (перегрев). Сделайте вывод о запасе.

---

## Часть C. RC-снаббер
1. На транзистор (сток-исток) добавьте RC-снаббер: `C_sn = 1 нФ`, `R_sn = 22 Ом`.
2. Сравните перенапряжение на ключе при выключении (без / со снаббером).
3. Оцените доп. потери в снаббере: `P_sn = 0.5·C_sn·U²·f_sw`.
   Пример: `0.5·1·10⁻⁹·100²·50·10³ = 0.25 Вт`.
