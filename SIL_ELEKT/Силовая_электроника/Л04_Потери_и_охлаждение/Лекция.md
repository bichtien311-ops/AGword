# Лекция 04. Потери, тепловые режимы и защита

**Модуль I. Основы и элементная база**

---

## 1. Зачем считать потери и тепло

Любые потери в приборе превращаются в тепло, нагревающее кристалл (переход). Если температура перехода `T_j` превысит предельную `T_j(max)` (обычно 125…175 °C), прибор деградирует или выходит из строя. Поэтому тепловой расчёт — обязательная часть проектирования: он связывает электрические потери с выбором системы охлаждения.

---

## 2. Классификация потерь

1. **Потери проводимости** — в открытом состоянии (`I_rms²·R_DS(on)` у MOSFET, `V_CE(sat)·I_ср` у IGBT, `U₀·I_ср + r_d·I_rms²` у диода).
2. **Потери переключения** — за время коммутации, `P_sw = (E_on + E_off)·f_sw`.
3. **Потери в драйвере** — на заряд/разряд затвора, `P_g = Q_g·V_GS·f_sw`.
4. **Потери покоя/утечки** — обычно малы, но важны в режиме standby.

Суммарные потери прибора:
```
P_total = P_cond + P_sw + P_g + ...
```

---

## 3. Тепловая модель: электрическая аналогия

Распространение тепла удобно моделировать как электрическую цепь:

| Тепловая величина | Электрический аналог |
|-------------------|----------------------|
| Мощность потерь `P` [Вт] | Ток `I` |
| Температура `T` [°C] | Напряжение `U` |
| Тепловое сопротивление `R_th` [°C/Вт] | Сопротивление `R` |
| Теплоёмкость `C_th` [Дж/°C] | Ёмкость `C` |

В установившемся режиме (как закон Ома):
```
ΔT = P · R_th     →     T_j = P · R_th(j-a) + T_a
```
где `T_a` — температура окружающей среды (ambient), `R_th(j-a)` — полное тепловое сопротивление «переход–среда».

---

## 4. Цепочка тепловых сопротивлений

Тепло от кристалла к воздуху проходит несколько барьеров, соединённых **последовательно**:
```
T_j ──[R_th(jc)]── T_корпус ──[R_th(cs)]── T_радиатор ──[R_th(sa)]── T_a
```
- `R_th(jc)` — переход–корпус (junction-case), задаётся производителем прибора.
- `R_th(cs)` — корпус–радиатор (case-sink), зависит от термопасты/прокладки и момента затяжки.
- `R_th(sa)` — радиатор–среда (sink-ambient), главный параметр выбора радиатора.

Полное сопротивление: `R_th(j-a) = R_th(jc) + R_th(cs) + R_th(sa)`.

**Температура перехода:**
```
T_j = P·(R_th(jc) + R_th(cs) + R_th(sa)) + T_a
```

---

## 5. Расчёт радиатора

Из условия `T_j ≤ T_j(max)` выражаем требуемое сопротивление радиатора:
```
R_th(sa) ≤ (T_j(max) − T_a)/P − R_th(jc) − R_th(cs)
```

**Вывод:** начинаем с `T_j(max) = P·R_th(j-a) + T_a`, переносим `T_a`, делим на `P`, вычитаем известные `R_th(jc)` и `R_th(cs)`. Если требуемое `R_th(sa)` получается отрицательным — пассивного радиатора недостаточно, нужен вентилятор или жидкостное охлаждение (либо снижение потерь).

### Численный пример
`P = 20 Вт`, `T_j(max) = 125 °C`, `T_a = 40 °C`, `R_th(jc) = 0.5 °C/Вт`, `R_th(cs) = 0.2 °C/Вт`.
```
R_th(sa) ≤ (125 − 40)/20 − 0.5 − 0.2 = 4.25 − 0.7 = 3.55 °C/Вт
```
Нужен радиатор с тепловым сопротивлением **не более 3.55 °C/Вт**.

---

## 6. Импульсные режимы: переходное тепловое сопротивление

При коротких импульсах мощности кристалл не успевает прогреться полностью — работает **теплоёмкость**. Вместо `R_th` используют **переходное тепловое сопротивление `Z_th(t)`**, которое для коротких импульсов меньше статического `R_th`.

**Описание графика `Z_th(t)`:** в логарифмическом масштабе кривая начинается с малых значений (короткие импульсы — малый нагрев) и выходит на горизонтальную асимптоту, равную статическому `R_th(jc)`, при длинных импульсах. Это позволяет приборам кратковременно выдерживать пиковую мощность, недопустимую в длительном режиме.

---

## 7. Защита силовых приборов

### 7.1. Снабберы (демпфирующие цепи)
- **RC-снаббер** — гасит высокочастотные колебания и ограничивает `dV/dt`.
- **RCD-снаббер** — ограничивает перенапряжение при выключении на индуктивную нагрузку; диод направляет энергию в конденсатор, резистор её рассеивает.

Энергия, запасаемая в конденсаторе снаббера: `E = 0.5·C·U²`; рассеиваемая мощность: `P_sn = E·f_sw`.

### 7.2. Ограничение dV/dt и di/dt
- `dV/dt` — снабберами и подбором драйвера;
- `di/dt` — последовательной индуктивностью / подбором скорости включения.

### 7.3. Защита от перенапряжений
- **TVS-диоды, варисторы** — поглощают выбросы.
- **Лавинные приборы** — выдерживают ограниченную энергию пробоя.

### 7.4. Защита от короткого замыкания
- **Десатурация (desaturation)** — для IGBT: контроль `V_CE` в открытом состоянии; аномальный рост `V_CE` означает КЗ → быстрое «мягкое» выключение.

---

## 8. Выводы

1. Потери = проводимость + переключение + драйвер; все они греют кристалл.
2. Тепловая модель — электрическая аналогия: `T_j = P·R_th + T_a`.
3. Радиатор выбирают из условия `T_j ≤ T_j(max)`, рассчитывая допустимое `R_th(sa)`.
4. Для импульсных режимов используют `Z_th(t)`; защиту реализуют снабберами, ограничителями и схемами десатурации.
